BioStor

Supplementary data and results for BioStor manuscript

Data and code

The following files are supplementary for the manuscript describing BioStor.

  • Contents of Tijdschrift voor Entomologie in RIS format
  • Scripts used to measure success in retrieving articles from BHL content

The file test.php reads the RIS file all.ris and attempts to find each article in BHL using the BioStor OpenURL resolver. The script outputs the article, if found, and on completion dumps three arrays:

  • For each year how many articles are in the RIS file
  • For each year how many articles were found
  • For each year how many articles had pagination in Roman numerals (BioStor doesn't support Roman numerals, hence didn't search for these references).

For each year the file also outputs an HTML listing each article found and displaying the corresponding page in BHL.
