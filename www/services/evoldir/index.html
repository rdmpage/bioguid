EvolDir via Twitter

The Evolution Directory (EvolDir) is a mailing list run by Brian Golding and used by evolutionary biologists to post announcements of conferences, jobs, software, and other events or developments of interest to the community.

Now you can follow EvolDir on Twitter to get regular updates. Note that to post to EvolDir you must join the EvolDir mailing list by sending an email to Golding[at]McMaster.CA.

The EvolDir Twitter service is being run by Rod Page (r.page[at]bio.gla.ac.uk). For details see this blog post.
