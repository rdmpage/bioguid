Openhandle

This is an instance of Tony Hammond's Openhandle application, which makes it possible to query Handles (including DOIs). For more details (and argument why this is useful, visit the project’s Googlecode homepage, or read the essays on the CrossTech blog.

The form below can be used to submit a query to the Openhandle application. Please specify the Handle ID for which you want to view the record (e.g. 10.1556/ABot.51.2009.1-2.2). To see some little used features of Handles, such as multiple resolution, try the DOI 10.2277/0521553385.

Optionally, you can also specify:

  1. A comma-separated list of Handle data types to limit your query to
  2. A comma-separated list of integer indexes from the record to return in the response
  3. The format in which the response should be returned
  4. A mimetype to use to return the response as (this is just a convenience feature for viewing responses in a web browser)
